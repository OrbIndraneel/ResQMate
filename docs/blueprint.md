# **App Name**: ResQMate

## Core Features:

- User Authentication & Profiles: Secure registration and login for both NGO Administrators and Volunteers, allowing each to create and manage their specific profiles (including NGO credentials, volunteer skills, and contact information).
- OTP Authentication: Implement a One-Time Password (OTP) system via email or SMS for enhanced security during user registration and login for both NGO administrators and volunteers.
- NGO Task Posting & Management: NGO Admins can post immediate and specific relief tasks, specifying details like required skills, number of volunteers, and urgency (normal/emergency). They can also view and manage their posted tasks.
- Volunteer Task Feed & Application: Volunteers can browse a personalized feed of available tasks relevant to their skills and current GPS location, then apply to tasks they are able to complete.
- Smart Task-Volunteer Matching: An AI-powered tool automatically matches available tasks to qualified volunteers based on their registered skills, real-time GPS location, and sends targeted notifications.
- Task Completion & Verification: System for NGOs to mark tasks as completed, potentially involving reviewing uploaded images of the completed work or scanning a QR code generated by the volunteer at the task site.
- Volunteer Recognition System: A system to track and display points earned by volunteers for each successfully completed task, visible on their dashboard.
- AI Task Description Assistant: A generative AI tool to help NGO Admins craft clear, concise, and comprehensive task descriptions based on minimal input, ensuring all critical requirements are communicated effectively.

## Style Guidelines:

- Primary color: A deep, professional teal-blue (#34535E) to convey reliability and seriousness in critical situations.
- Secondary color/Background elements: A balanced teal (#2B7B7B) for UI components like cards, panels, or secondary calls to action, providing visual structure.
- Accent color: A vibrant, clear teal-green (#57A498) for highlighting important information or less urgent interactive elements.
- Call to action/Urgency color: A bright, impactful red (#CF223B) for critical buttons like 'claim task' and urgent alerts to signify immediate action.
- Warning/Error color: A rich, deeper red (#90112F) for error messages, warnings, or states requiring serious attention.
- Body and headline font: 'Inter', a modern grotesque sans-serif for its exceptional legibility and neutral, objective appearance, ideal for presenting critical information clearly and efficiently.
- Utilize simple, universally recognizable icons that quickly convey meaning and task categories, ensuring immediate understanding even under stressful conditions.
- Clean and functional layout prioritizing clarity and efficiency. Task-oriented design with clear calls to action to guide users quickly through essential workflows for rapid response.
- Subtle, quick, and responsive animations to provide instant feedback for user interactions (e.g., task claiming, notifications), maintaining a sense of urgency without being distracting.
